Windows CMD [已退出进程，代码为 1 (0x00000001)]

本文最后更新于：2024年1月14日 中午

问题描述

卸载 Anaconda 后， 打开 CMD(命令提示符) 时，出现如下错误：

Windows PowerShell 正常

环境

OS: Win11

解决方案

  1. 打开注册表 win + r 输入 regedit
  2. 将 HKEY_CURRENT_USERProcessor 中的 AutoRun 赋值为空
  3. 将 HKEY_LOCAL_MACHINEProcessor 中的 AutoRun 赋值为空（我这里没有这个键值）
  4. 问题解决
